Game Controls and Status

Use arrow keys to move left/right, up to jump, and space to rotate tunnel. Collect power cells to unlock characters.

How the Run 3 Endless Runner Game Works

This game simulates an infinite run through space tunnels, where players must calculate trajectories, timing for jumps, and rotations to survive. The procedural generation ensures each playthrough is unique, challenging spatial reasoning and quick math skills like estimating distances and angles.

To play, select a character and enter infinite mode. The game uses 3D rendering to create immersive tunnels with gaps and walls. Physics engine handles gravity and momentum for realistic movements.

Understanding Game Mechanics and Physics

The core mechanics involve vector calculations for movement and collision detection. For example, jumping follows a parabolic path defined by:

\[ y = y_0 + v_0 t - \frac{1}{2} g t^2 \]

Where \( g \) is gravity, \( v_0 \) initial velocity. Rotations alter the coordinate system, teaching transformation matrices in 3D space.

Step-by-Step Guide to Playing

1. Click 'Start Game' to initialize the 3D scene.

2. Use left/right arrows to change lanes in the tunnel.

3. Press up arrow to jump over gaps.

4. Spacebar rotates the tunnel for wall-running.

5. Collect power cells to increase score and unlock new characters like the Skater for better handling.

Advanced tip: Calculate optimal jump timing by observing tunnel speed, which increases over time.

This game applies concepts from geometry, physics, and probability. Procedural tunnel generation uses algorithms like Perlin noise for random yet fair challenges. Players implicitly learn about vectors and transformations while navigating.

FAQ

How do I unlock new characters? Collect power cells in infinite mode to unlock options like the Rabbit for higher jumps.

What happens if I fall? Game over, but you can restart and use collected cells to continue with a different character.

Is the game mobile-friendly? Yes, controls adapt to touch for jumping and swiping.

Can I play in explore mode? This version focuses on infinite mode, but levels can be simulated by progressing further.
