How the What If I Bought Bitcoin Calculator Works

This calculator allows users to input an investment amount in USD (or equivalent in selected currency) and a past date to simulate purchasing Bitcoin at that time's market price. It fetches historical Bitcoin prices from reliable APIs and compares them to the current price to compute returns.

The core calculation determines how many Bitcoin (BTC) units could have been bought with the invested amount on the chosen date. Then, it multiplies that BTC amount by the current Bitcoin price to find the present value. Profit or loss is the difference between current value and initial investment, expressed both in absolute terms and as a percentage.

Key Formulas Used

BTC Bought = Investment Amount / Historical Bitcoin Price

Current Value = BTC Bought × Current Bitcoin Price

Profit/Loss = Current Value - Investment Amount

Percentage Return = (Profit/Loss / Investment Amount) × 100

These are rendered mathematically as:

\[ \text{BTC Bought} = \frac{\text{Investment Amount}}{\text{Historical Price}} \] \[ \text{Current Value} = \text{BTC Bought} \times \text{Current Price} \] \[ \text{Profit/Loss} = \text{Current Value} - \text{Investment Amount} \] \[ \text{Percentage Return} = \left( \frac{\text{Profit/Loss}}{\text{Investment Amount}} \right) \times 100 \]
